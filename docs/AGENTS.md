# AGENTS.md

## Scope

This folder contains project documentation for developers and AI agents.

## Structure

- `architecture.md`: visao geral da stack, estrutura de pastas, fluxo de dados, convencoes de rotas.
- `database.md`: schema, enums, tabelas, IDs, migrations, organizacao de queries e actions.
- `auth.md`: Better Auth, sessao, middleware, ownership, roles, autorizacao.
- `publishing.md`: fluxo de publicacao, geracao de HTML, serving por subdominio, storage.
- `editor.md`: integracao GrapesJS, blocos, salvamento, assets, preview.
- `deploy/dokploy.md`: guia de deploy com Dokploy em VPS.
- `deploy/nginx.md`: configuracao Nginx para wildcard subdomains.

## Rules

- Keep docs concise and accurate.
- Update docs when architecture, schema, auth flow, publishing model, or environment variables change.
- Do not document secrets or real credentials.
- Use Portuguese for content, English for AGENTS.md files.
